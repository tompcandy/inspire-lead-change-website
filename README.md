# Inspire Lead Change — website

Coaching-led consultancy site for Inspire Lead Change (ILC), based in Haarlem, Netherlands.

Static HTML/CSS/JS site, designed to deploy on GitHub Pages and serve `www.inspireleadchange.com`.

## Pages

- `index.html` — homepage
- `what-we-do.html` — three pillars (Operational Excellence, Leadership Development, Performance Dashboards)
- `our-approach.html` — Route to Excellence (six-step methodology)
- `about.html` — Tom Candy bio + companies supported
- `contact.html` — contact form + info

## Tech

- Plain HTML, CSS, and a small `script.js` for the mobile nav, contact-form mailto wiring, and entrance animations
- Google Fonts (Playfair Display, Inter)
- No build step, no framework, no dependencies

## Local preview

Open `index.html` in a browser. That's it.
